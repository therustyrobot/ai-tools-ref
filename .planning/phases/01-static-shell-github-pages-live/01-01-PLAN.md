<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → GitHub Pages CDN | User fetches static HTML; no server processing |
| Browser → Google Fonts CDN | Font and icon stylesheet loaded from third-party CDN |
| Browser → Tailwind CDN | Framework script loaded from third-party CDN |
| Static HTML → External links | All 12 repo links navigate to github.com |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Tampering | cdn.tailwindcss.com, fonts.googleapis.com | accept | Third-party CDN integrity is out of scope for a public static reference page; standard industry practice for this use case; revisit in V2 if CSP headers are needed |
| T-01-02 | Denial of Service | GitHub Pages availability | accept | GitHub Pages SLA covers public repos; no mitigation needed at this scale |
| T-01-03 | Elevation of Privilege | All `<a target="_blank">` repo links | mitigate | Every external link must include `rel="noopener noreferrer"` — prevents reverse tabnabbing; enforced in Task 2 action and verified by automated grep check |
| T-01-04 | Information Disclosure | docs/index.html content | accept | All data is intentionally public (GitHub star counts, repo metadata); no PII, no secrets; static file fully public by design |
</threat_model>
